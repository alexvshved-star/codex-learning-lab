Контекст проєкту

ALTACO — українська компанія з Києва, яка займається дистрибуцією якісного італійського натурального каменю та кварцового агломерату. Представляє бренди Bagnara та Santa Margherita в Україні.

Матеріали: мармур, граніт, кварцит, травертин, онікс, кварцовий і мармуровий агломерат.

ALTACO — реальний бізнес-полігон для тестування AI-рішень: каталог, склад, клієнтські запити, комерційні пропозиції, логістика.

EONYX

EONYX створює прикладні AI-системи для B2B-компаній реального сектору. Фокус — не абстрактні чат-боти, а контрольовані AI-агенти, які працюють із реальними бізнес-даними.

Напрями: автоматизація процесів, кастомні AI-агенти, цифровізація експертизи, інтеграція AI у продажі, склад, документообіг і аналітику.

ALTACO є практичним майданчиком для EONYX, де AI-рішення тестуються на реальних задачах.

Три кити вебу

🏗️HTML

HTML — це структура сторінки. Як скелет: заголовки, параграфи, кнопки, зображення.

🎨CSS

CSS — це оформлення. Кольори, шрифти, відступи, анімації. HTML без CSS виглядає як текст без дизайну.

⚡JavaScript

JavaScript — це поведінка. Завдяки йому кнопки щось роблять, тема перемикається, чекліст рахує прогрес.
