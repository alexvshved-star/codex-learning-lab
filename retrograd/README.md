# Ретроградный Меркурий · Центр космического контроля

Интерактивный мем-артефакт: радарный монитор космического центра «обнаруживает»
ретроградный Меркурий (29.06 — 27.07) как боевую цель. Оммаж на монтажный
running-gag из «Austin Powers: The Spy Who Shagged Me» и mission-control эстетику
«Аполлона-13» — весь текст и графика оригинальные, без цитат из сценариев.

## Что внутри
- Круглый радар на `<canvas>`: скан-луч, кольца дальности, летящий объект `☿ RETRO`.
- Эквалайзер-полоса канала связи.
- Боевой чат, который печатается сам: реплики обрываются перед «очевидным словом»
  и переключаются на следующий пост — как склейки в оригинале.
- Кнопка **Запустить перехват** активируется после вступительных реплик.

Чистый ванильный HTML/CSS/JS, без зависимостей. Один файл — `index.html`.

## Локально
Открыть `index.html` в браузере.

## Деплой
GitHub Pages: Settings → Pages → Deploy from a branch → `main` / `root`.
