# Retrograde Intercept

A single-file browser toy: a space-control radar meme that plays out a cut-away
running-gag (Mercury retrograde "detected" as a target), then unlocks a small
wave-shooter. Trilingual (RU / UK / EN), vanilla HTML/CSS/JS, no dependencies.

## Flow
1. Intro comms type themselves → **Start intercept** unlocks.
2. The running-gag plays to the end (it can't be skipped — **DESTROY** stays locked).
3. At the finale, **DESTROY** unlocks and launches the game:
   ship at the bottom, descending `☿` targets, waves, 3 lives.

## Controls
- Desktop: ← → to move, SPACE to fire.
- Mobile: on-screen ◀ / FIRE / ▶.

## Language
RU / UK / EN switch in the header (resets to the intro in that language).

Single file — `index.html`. All text and graphics are original.
